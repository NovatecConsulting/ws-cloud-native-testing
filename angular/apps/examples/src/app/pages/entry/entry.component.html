Welcome to the basic examples for Angular Testing

This app includes basic examples on how to write tests for angular applications. It will grow with time and showcase more and more scenarios, so you can use it as a reference for your testing needs.

Just a little FAQ:

This repository is mocking a lot! Is this a good way of doing things?

This is a first impression for the first set of examples in this application. The Weather Page is tested with a lot of mocking except for scenarios, where this is not applicable. Further it provides a small use case end-to-end test and an integrative test for the, main page to protect us while refactoring large parts.

We will provide more examples with other approaches later.

 Is there anything I should know before visiting this app?

We suggest that you read the basic introduction to testing first. It provides an overview as to why we write tests and how we should do so. There are many examples explained, much more than this app contains. For that reason, this repository will be growing over time.

I do not find examples for my specific problem, where can I get help?

Just reach out to us. If your example is already covered, or something close to it, we will point you to it. Otherwise we have the chance to improve upon our examples. But please keep in mind, this is a show case for the testing basics. Thus, we can not ensure your topic to be introduced.
